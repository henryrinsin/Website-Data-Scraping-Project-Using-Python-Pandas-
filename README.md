# Website-Data-Scraping-Project-Using-Python-Pandas-By Uchechukwu Okpara
This project demonstrate how to scrape data from a website using Python. The script collects public available information from the site "https://en.wikipedia.org/wiki/List_of_largest_companies_in_the_United_States_by_revenue", processed it into a structured format, and export the results for further analysis
Features:
Scrapes data from https://en.wikipedia.org/wiki/List_of_largest_companies_in_the_United_States_by_revenue using Python
Parses HTML content efficiently
Cleans and structures the extracted data
Exports the data to a CSV file
Includes basic error handling for failed requested

Technologies Used:
Python 3,
Requests,
BeautifulSoup4,
Pandas.
